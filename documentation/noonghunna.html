Noon Ghunna

Since Unicode 7 (June, 2014), Unicode documentation[1] says that U+06BA ARABIC LETTER NOON GHUNNA should be dotless in all four contextual forms. In cases where nasalization needs to be marked in initial or medial contexts, the ordinary noon, U+0646 ARABIC LETTER NOON, should be used, optionally with U+0658 ARABIC MARK NOON GHUNNA.

1 The Unicode Consortium. The Unicode Standard, Version 14.0.0, (Mountain View, CA: The Unicode Consortium, 2021. ISBN 978-1-936213-29-0), pg 391.
